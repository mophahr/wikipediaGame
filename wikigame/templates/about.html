{% trans "Rules of the Game" %}

{% trans "For each challenge, the goal is to move with the fewer number of clicks from the starting person to a target person. The links available on each step correspond to the hyperlinks contained in the (German) Wikipedia page of that person that point to a Wikipedia page of another person. "%}

{% trans "Concepts" %}

{% trans "This simple game illustrates some scientific ideas studied in the field of complex systems:"%}

{% trans "Network"%}

{% trans "A network (or graph) is composed by nodes connected to each other by links. In the example of the game, the nodes are (Wikipedia pages of) people and the links are (hyper) connections between them. Networks appear in social (e.g., diesease spreading), biological (e.g., food webs), and technological (e.g., the Internet) applications and are important because the connectivity of the network is directly connected with the behavior (efficiency) of the system. "%}

{% trans "Small-world networks"%}

{% trans " A typical property of (social) networks is that nodes are connected to each other in groups (friends of my friends are also friends). At the same time, two arbitrary nodes in the network are typically close to each other. More precisely, in a network with N nodes (very large), the typical distance between nodes grows slowly with N (typically with the logarithm of N). Networks which satisfy these properties are known as small world networks. In social networks this effect is sometimes known as six degrees of separation "%}

{% trans "Navigability"%}

{% trans " The fact that there is a short path between two nodes of a small world network does not imply that it is easy to find it! This game illustrates this problem of navigability (or routing) using only local information and some expectation about the target node. "%}
